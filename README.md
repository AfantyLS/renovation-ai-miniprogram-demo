# AI 装修方案生成器微信小程序 Demo

这是一个最小可运行的微信小程序 Demo，用来验证“用户上传户型图 + 填写装修需求 + 生成装修方案报告”的核心流程。

第一版不接真实大模型、不做复杂 3D、不做支付和登录，重点是把产品主链路跑通。

## 当前功能

- 首页展示产品入口和 Demo 能力
- 创建装修方案
- 选择户型图
- 填写城市、面积、预算、风格、家庭结构和重点需求
- 使用本地 mock 逻辑生成装修报告
- 查看整体方案、房间方案、材料清单、家具家电清单、预算建议和下一步任务

## 推荐开发工具

- 微信开发者工具
- 基础库建议使用较新版本
- AppID 可先使用测试号或游客模式

## 如何运行

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择本仓库根目录。
4. AppID 可以使用测试号，或选择无 AppID 调试。
5. 编译运行。

## MVP 开发原则

```text
先打通主链路，再接真实 AI。
先用 mock 数据生成报告，再替换为云函数或后端 API。
先做小程序端体验，再扩展 HTML/PDF 完整报告。
```

## 后续接入真实 AI 的建议

小程序端不应该直接保存大模型 API Key。推荐架构：

```text
微信小程序
  -> 微信云函数 / 自建后端
  -> AI 服务
  -> 返回结构化装修报告 JSON
  -> 小程序渲染报告
```

后续可以新增：

- 微信登录
- 项目列表
- 报告保存
- 真实 AI 报告生成
- 图片生成
- HTML 报告分享链接
- PDF 下载
- 设计师人工优化入口
